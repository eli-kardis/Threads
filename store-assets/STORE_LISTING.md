# Chrome Web Store Listing Guide

## Extension Name
**Threads to Notion Sync**

## Short Description (132 characters max)
Threads 게시글을 자동으로 Notion에 동기화. 아이디어 손실 방지 & 창작 시간 확보!

## Detailed Description

**Threads에 작성한 아이디어, 놓치고 계신가요?**

Threads to Notion Sync는 Threads에 작성한 글을 자동으로 Notion 데이터베이스에 저장해주는 크롬 확장 프로그램입니다.

### 주요 기능

**자동 동기화**
- Threads에 새 글을 올리면 자동으로 Notion에 저장
- 설정 가능한 동기화 간격 (5분~60분)
- 수동 동기화 버튼으로 즉시 동기화

**해시태그 필터링**
- 특정 해시태그가 포함된 글만 동기화 (포함 모드)
- 특정 해시태그가 있는 글 제외 (제외 모드)

**통계 대시보드**
- 일별/주별/월별 동기화 현황 차트
- 성공률 통계
- 최근 동기화 기록

**유연한 필드 매핑**
- Notion 데이터베이스 필드에 자유롭게 매핑
- 제목, 본문, 이미지, 태그, 작성일, 원본 URL 지원

### 이런 분들께 추천합니다
- Threads를 아이디어 메모장으로 사용하시는 분
- 소셜 미디어 콘텐츠를 체계적으로 관리하고 싶은 분
- Notion으로 지식을 정리하시는 분
- 창작자, 마케터, 기획자

### 설정 방법
1. Threads API 토큰 발급 (Meta Developer 계정 필요)
2. Notion Integration 생성 및 시크릿 키 발급
3. Notion 데이터베이스 생성 및 ID 확인
4. 확장 프로그램 설정에서 연결 완료!

### 개인정보 보호
- 모든 데이터는 사용자 브라우저에 로컬 저장
- 서버 전송 없음, 광고 없음, 추적 없음
